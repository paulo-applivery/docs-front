title: Firewall Configuration on macOS
description: Manage the macOS application firewall and allowed services.
updatedDate: 2024-08-12
---

Use profiles to configure the macOS application firewall and define allowed inbound services:

- Enable the firewall and stealth mode for stronger security
- Control app exceptions and signed software behavior
- Audit changes and enforce policies via MDM

Test configurations with representative workloads to avoid breaking legitimate access.
