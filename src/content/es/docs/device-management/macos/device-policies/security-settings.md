---
title: macOS Security Settings
description: Establish security baselines including password policies and privacy controls.
updatedDate: 2024-08-12
---

Define a baseline for macOS devices using configuration profiles and restrictions:

- Password policies (length, complexity, rotation)
- Privacy preferences policy control (PPPC) for app permissions
- Screen lock, idle timers, and login window settings

Review and audit settings regularly to ensure compliance and minimize user disruption.
